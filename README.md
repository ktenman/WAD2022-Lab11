# WAD2022-Lab11


## Pre-requisites
Node.js \& NPM

## Clone the repository
```
git clone https://github.com/M-Gharib/WAD2022-Lab11.git
```

## Navigate to the project directory
```
cd <project name>
```

## Install dependencies
```
npm install
```

## Run the project
```
npm start
```

or 
```
npm server.js
```
